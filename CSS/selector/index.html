CSS selector precedence

The most specific selector is used.(between id and class id css will be applied)

Defaults to the last rule in the stylesheet(css for selector is written in two different places then the second or least taken in place will be applied)

precednce for css

  1. Inline css:1000pts
  2. ID :100pts
  3. class :10pts
  4. Pseudo elemets :10pts
  5. Attributes :1pt
  6. Elements :1pt
  7. Pseudo elements: 1pt
