Various units used in css

length

  1. PX: pixels,absolute length(Which is not dependent on something)
  2. -em: relative to font size(dependent unit)
  3. -rem(dependent unit): relative to root element font size(closest relative value)
  4. -vw: 1% of viewport width
  5. -vh: 1% of viewport height
  6. -ch: width of "O"th character in html(especially in paragraphs which contains 50-70 characters)
  7. -%: Percentage,usually relative to parent value

margin / padding

  1. -rem: for closer to absolute value(responsive website)
  2. -em: to scale with root element
  3. -px: for small values last resort

Borders,shadows

  1. -px: for small values
  2. -rem/em: okay here too-->scaling doesnot always look great

Font size

  • -rem: is usually best
  • -em: to scale to parent size
  • -px: as a last resort

Colors

  • -keywords: red,blue,green,etc
  • -HEX RGB: #4B70AF(amount of red,green,blue)
  • RGB:-RGBA(75,15,175,0.5)
  • HSL: Hus,saturation and lightness(hsl(210,40%,49%,0.8))

Inline VS block element

Inline element don't have width and height(It can't take css width and height properties)

To change inline element to block element use Display:block

If block to inline use display:inline property

Display:inline:block==>It doesn't go to next line but we can define the css properties (Widths and heights)(Images contains inline-block property)
